Generic utilities

CMB.quadprod — Function
quadprod(A, b, n, dir=:col)

Computes the quadratic product $ABA^\top$ efficiently for the case where $B$ is all zero except for the nth column or row vector b, for dir = :col or dir = :row, respectively.
